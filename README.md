# SYSU 计算复杂性课程编程作业

本仓库包含中山大学计算复杂性课程的编程作业。

## 项目结构

```
SYSU_Computational_Complexity/
├── README.md                    # 项目总览说明
├── .gitignore                   # Git忽略配置
├── lab1/                        # 第一次作业
│   ├── README.md
│   ├── src/                     # 源代码
│   ├── docs/                    # 文档
│   └── test/                    # 测试文件
└── lab2/                        # 第二次作业
    ├── README.md
    ├── src/                     # 源代码
    │   ├── models/              # 数据模型
    │   ├── parsers/             # 解析器
    │   ├── reductions/          # 归约算法
    │   └── utils/               # 工具函数
    ├── tests/                   # 测试文件
    └── docs/                    # 文档
```

## 作业说明

### Lab1: 通用图灵机
实现多带图灵机的编码和通用图灵机模拟器。

**主要功能：**
- 多带图灵机的编码算法
- 通用图灵机(UTM)模拟器
- 命令行交互界面

### Lab2: NP完全性归约
实现3SAT到节点覆盖(Vertex Cover)和独立集(Independent Set)的归约算法。

**主要功能：**
- CNF公式解析器
- 图结构表示
- 3SAT到Vertex Cover归约
- 3SAT到Independent Set归约
- 可视化工具
